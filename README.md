# Project Title: E-Mart-Sales Analysis Dashboard

# Short Description :
i) Purpose: Tracks e-commerce performance across regions.
ii) Goal: Optimizes sales strategies using data insights.
iii) Focus: Analyzes profitability, customer behavior, and payments

# Tech Stack :
i) Power BI: For data modeling and visualization.
ii) Power Query: For data transformation and cleaning.
iii) DAX: For custom metrics and KPI calculations

# Data Source :
i) E-Commerce Dataset: Contains retail sales transactions data.
ii) Variables Used: Includes fields for revenue, profit, categories, states, and payment modes.

# Features & Highlights :
i) KPI Matrix: Displays key metrics like total revenue, profit (37K), and quantities (6K).
ii) Geographical Breakdown: Visualizes top-performing regions like Maharashtra.
iii) Product Breakdown: Reveals segment performance across Clothing, Electronics, and Furniture.
iv) Trend Analysis: Tracks monthly profit fluctuations to find seasonality.
v) Dynamic Slicers: Filters entire report by specific quarters instantly.

# Business Problems Solved :
i) Loss Tracking: Identifies unprofitable months like May and December.
ii) Inventory Control: Matches stock levels with category demand percentages.
iii)Payment Optimization: Tracks preference shifts toward Cash on Delivery (44%).

# Business Impact & Insights :
i) Dominant Category: Clothing drives the highest transaction volume at 63%.
ii) Regional Leader: Maharashtra leads all other states in total sales amount.
iii) Profitable Niches: Printers generate the highest profit among sub-categories.
iv) Seasonal Drops: Summer months show significant profit dips needing promotions.
